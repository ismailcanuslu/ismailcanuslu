# 💫 Hakkımda:
.NET ekosisteminde modern yazılım mimarileri (Clean Architecture, N-Layer, MVC gibi) ve sürdürülebilir kod prensipleri (SOLID) doğrultusunda ölçeklenebilir backend çözümleri üretmekteyim. Yazılım geliştirme süreçlerini; Linux sistem yönetimi, Proxmox sanallaştırma ve Docker orkestrasyon yetkinliklerimle birleştirerek uçtan uca altyapı ve bulut mimarileri kurgulamaktayım.<br><br>GitHub/GitLab CI/CD hatları üzerinden otomatize ettiğim operasyon süreçlerini, üretim ortamında gözlemlenebilirlik (Trace, Metric, Log) metotlarıyla önleyici bir yaklaşımla yönetmekteyim. Sektörel inovasyonları ve modern mühendislik standartlarını yakından takip ederek, teknik yetkinliklerimi güncel teknolojiler doğrultusunda sürekli geliştirmekteyim.
